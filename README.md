# Task Manager App

## Description
A Python app that lets users manage tasks with:
- Add new tasks
- Edit existing tasks
- Mark tasks as completed

Tasks are stored persistently in a JSON file so they remain available even after the program exits.

---

## Technologies
- Python
- JSON (for storage)
